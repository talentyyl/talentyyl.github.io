Entity Framework

发表于 2019-05-09
  • 什么是 EF

EF 是 Entity Framework 的简称，是微软 Object Relational Mapper（对象关系引射器），也就是我们平常所说的 ORM，它使得 .NET 开发者可以使用特定的领域对象处理关系数据，它消除了开发人员通常需要编写的大多数数据访问代码的需要。

  • 使用 Entity Framework 的一些好处

    • 因为开发者不需要为数据访问编写素有需要的 ADO.NET 管道代码，因此节省了很多开发时间
    • 我们可以使用更高级的语言（例如 C#）来编写所有的数据访问逻辑而不是编写 SQL 查询和存储过程
    • 因为数据库表没有高级关系（例如继承），然而领域实体是可以有的，所以业务模型（也就是概念模型）可以使用实体间的关系来适配应用领域

    EF 是建立在 ADO.NET的框架之上的，它的底层仍是使用 ADO.NET 的方法和类来执行数据操作
